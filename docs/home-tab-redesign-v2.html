✦ Correções da v1 → v2 (o que eu havia inventado)

  • "~45 min" no treino → não existe duração estimada; só "5 exercícios" (duração só após concluir).
  • "12 semanas de sequência" no topo → seu streak real é < 2 → conquista bloqueada ("Comece um streak").
  • Herói fixo "Iniciar treino" → herói é máquina de estados ligada ao dia do calendário (ver faixa abaixo).
  • Prontidão sempre presente → é opcional: some sem wearable (HR/HRV "–" no seu print).
  • Conquistas usam dados reais: streak bloqueado · 3 treinos · 3/25 rumo a marco.
  • Prontidão mantém a recomendação completa ("regular/reduzido" têm 2–3 frases de coaching).
