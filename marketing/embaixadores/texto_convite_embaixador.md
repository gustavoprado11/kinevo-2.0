## Como gerar o convite personalizado

Para criar um PDF com o nome do treinador, rode o script:

```bash
python gerar_convite_embaixador.py "Nome do Treinador" "Convite_NomeDoTreinador.pdf"
```

Exemplos:
```bash
python gerar_convite_embaixador.py "Rafael Oliveira" "Convite_Rafael_Oliveira.pdf"
python gerar_convite_embaixador.py "Ana Costa" "Convite_Ana_Costa.pdf"
```

> Nota: o script requer Python com a biblioteca `reportlab` instalada (`pip install reportlab`).
